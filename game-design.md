## 核心架构

```
单文件 index.html
├── CSS（内联 <style>）
│   ├── CSS 变量 :root（主题色、品质色、稀有度色）
│   ├── 页面布局（TopBar / Content / BottomNav）
│   └── 各子系统组件样式
├── HTML 结构
│   ├── #page-soul        武魂页
│   ├── #page-hunt        试炼/狩猎页
│   ├── #page-lottery     星运页
│   ├── #page-tasks       修炼/任务页
│   ├── #page-abyss       世界探索页
│   ├── #PF               魂环融合子页
│   ├── #grimoire-sidebar 武魂图鉴侧边栏
│   └── 全局 Overlay（结果/模态框/GM面板/战斗）
└── JavaScript（内联 <script>）
    ├── 常量定义（QC / SD / RT / BONE_POOL / ARTS / …）
    ├── 游戏状态 G（localStorage: dlv3）
    ├── 图鉴状态 GRIMOIRE（localStorage: dlv3-grimoire）
    └── 功能函数（按系统分组）
```

**存档机制**
- 主存档：`localStorage['dlv3']` — JSON序列化的 `G` 对象
- 图鉴存档：`localStorage['dlv3-grimoire']` — 跨重置永久保留
- 自动保存：每 10 秒 + 每次关键操作后调用 `saveG()`

## 游戏状态

`G` 对象是游戏的单一数据源，包含以下核心字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `sp` | number | 当前魂力（主货币） |
| `exp` | number | 当前等级内经验值 |
| `level` | number | 玩家等级 (1~200+) |
| `soul` | object\|null | 当前武魂对象 |
| `equippedBones` | object | 已装备魂骨 `{slotKey: boneObj}` |
| `equippedArt` | object\|null | 已装备神器 |
| `bag` | array | 背包物品列表 |
| `titles` | array | 拥有的称号列表 |
| `equippedTitle` | object\|null | 当前装备的称号 |
| `tasks` | array | 当前任务列表 |
| `lotTotal` | number | 总抽取次数 |
| `lotTopCount` | number | 顶级奖池累计抽取次数（彩蛋触发器） |
| `cosmicOwned` | boolean | 是否拥有宇宙之核（唯一道具） |
| `extraPower` | number | 额外战力加成 |
| `godTrials` | array | 成神之路试炼状态 |
| `godChoice` | string\|null | 选择的神位考核 ID |
| `godExamProgress` | object | 各神位考核通关进度 |
| `abyss` | object | 异界副本状态（令牌/碎片/进度） |
| `seasonal` | object | 踏春欧皇活动状态 |
| `powerMilestones` | array | 已达成的战力里程碑 |

### 等级经验公式

```
expForLv(lv):
  lv <= 10: 需要 200 EXP
  lv > 10:  需要 200 + (lv - 10) * 100 EXP
```

### 战力计算公式

```
calcPower():
  基础 = level × 80 + extraPower
  + 武魂品质基础战力 × 品质倍率 × 80
  + 所有装备魂环的 pw 值之和
  + 所有装备魂骨的 pw 值之和
  + 神器 pw × 神器品质倍率 (mul)
    + 若有神赐魂骨：再加 basePw × mul × 0.5 × 神骨数量（共鸣）
  + 已装备称号的 pw 值
```

## 武魂系统

### 品质分级（QC）

| 品质 key | 名称 | 基础概率 | 战力倍率 |
|----------|------|----------|----------|
| `common` | 普通 | 18% | ×1 |
| `rare` | 稀有 | 24% | ×1.5 |
| `epic` | 史诗 | 21% | ×2 |
| `legend` | 传说 | 17% | ×3 |
| `apex` | 顶级 | 8% | ×5 |
| `hc` | 普通隐藏 | 4% | ×2.5 |
| `ha` | 顶级隐藏 | 2% | ×4 |
| `twin` | 双生武魂 | 4% | ×6 |
| `triple` | 三生武魂 | 0.5% | ×10 |

> 武魂总数：101 种（分布于 SD 对象各品质池）

### 武魂数据结构 (SD)

每个武魂条目：
```js
{
  n: "名称",
  i: "图标emoji",
  d: "描述",
  a: ["属性1", "属性2"],
  p: 初始魂力等级,
  tw: true,   // 仅双生
  tri: true,  // 仅三生
}
```

### 觉醒流程

1. 按品质权重随机选取品质 `getQK()`
2. 从对应品质池随机选取武魂
3. 6% 概率触发属性变异，追加稀有属性词条
4. 生成初始魂技 `genSkills()`
5. 展示结果界面（OR overlay）
6. 关闭结果后进入武魂主页

### 二次觉醒

- 消耗：500 魂力
- 22% 概率升为神级（quality → apex，divine=true）
- 28% 概率追加暴击属性
- 42% 概率品质等级提升
- 成功后记录图鉴

## 魂环系统

### 魂环等级（RT）

| 名称 | 年份 | 战力(pw) | 特殊 |
|------|------|----------|------|
| 百年 | 100 | 80 | — |
| 千年 | 1,000 | 200 | — |
| 万年 | 10,000 | 500 | — |
| 十万年 | 100,000 | 1,200 | 速度+15/攻+20/防+10 |
| 百万年 | 1,000,000 | 3,500 | 速度+30/攻+50/防+25/魂力回复+10 |
| 不可估量 | ~10亿 | 15,000 | 攻击忽视防御30% |
| 神赐 | ~百亿 | 50,000 | 受致死攻击概率免疫 |
| 宇宙之核 | 1e14 | 999,999 | 唯一·无视一切防御 |

### 装配规则

- 武魂最多 10 个魂环
- 第 N 个魂环需要 Lv.(N×10)
- 装配时自动生成对应魂技
- `不可估量` 装配额外触发质变魂技（概率）
- `宇宙之核` 全局唯一，已持有时改为神赐

### 低阶魂环分解

百年/千年/万年魂环可在装配界面分解：
```
分解经验 = floor(pw × 0.5 + 50)
```

## 战斗系统

回合制，最多 **8 回合**。

### 角色属性

**玩家**（基于当前游戏数据动态生成）：
- 最大HP = 战力 × 2 + 等级 × 500
- 攻击 = 战力 × 0.8 × (1 + 魂环攻击加成)
- 速度 = 10 + 魂环速度加成/10
- MP = 100（最大），普攻回复15，防御回复25

**敌方**：固定数值，BOSS 有3阶段变形。

### 行动选项

| 行动 | 效果 |
|------|------|
| 普通攻击 | 基础攻击伤害，回复15MP |
| 防御 | 减伤40%，回复25MP |
| 魂技 (×4) | 1.5~2.7倍攻击，消耗20~50MP，附带状态异常 |

### 六种状态异常

| 状态 | 效果 |
|------|------|
| 中毒(poison) | 每回合损失3%最大HP |
| 强毒(poison2) | 3%HP + 额外层数 |
| 燃烧(burn) | 每回合损失4%最大HP |
| 减速(slow) | 攻击力×0.7 |
| 冰封(freeze) | 跳过该角色本回合行动 |
| 虚弱(weaken) | 己方攻击×0.8/受到伤害×1.2 |

### 双生/三生融合爆发

- 双生武魂使用第4技能额外触发融合爆发：**×1.8** 额外伤害
- 三生武魂额外触发三重融合：**×2.5** 额外伤害

## 魂环融合系统

- 消耗2个魂环，可选加入0~3种药草
- 基础成功率：**50%**
- 药草成功率加成：依药草 `fb.s` 值（总计最高+45%）
- 基础质变概率：**1%**（质变=品级提升1阶）
- 药草质变加成：依药草 `fb.m` 值（最高+50%）
- 失败：两个魂环同时消耗
- 成功：产出等级 = max(A阶, B阶)；质变则+1阶

## 彩蛋系统

**触发条件**：顶级奖池累计抽取 **≥ 1001 次**（`G.lotTopCount >= 1001`）

奖励：获得武魂「初」（∞，三生品质），初始战力10,000,000，装备全部重置。

武魂「初」特性：
- 品质：triple（三生）
- divine: true，isChu: true
- 初始魂技：宇宙归一 / 万物溯源 / 虚空之眼
- 战力加成：10,000,000 extraPower
